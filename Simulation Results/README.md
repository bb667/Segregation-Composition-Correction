## Simulation Results

Full Schelling simulation outputs are large and are hosted externally.

The folders `Interaction_*_Different_Composition_Reduced` and `Non_Interaction_Different_Composition_Reduced` contain grid snapshots sampled between 500,000 and 2,000,000 simulation steps in increments of 20,000 steps. These late-stage snapshots capture equilibrium spatial structure and were used for the analysis and figures in the paper.

Raw full trajectories are not archived due to file size.

Simulation data can be downloaded from:
